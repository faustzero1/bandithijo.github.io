Konfigurasi yang saya gunakan adalah.

```
$ sudo vim /etc/X11/xorg.conf.d/20-intel.conf
```
```
Section "Device"
  Identifier  "Intel Graphics"
  Driver      "intel"
  Option      "DRI"           "2"
  Option      "AccelMethod"   "uxa" # fallback
```
