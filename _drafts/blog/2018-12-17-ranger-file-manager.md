# Prakata

Ranger adalah *text-based file manager* (aplikasi untuk memanajemen file bebasis teks) yang ditulis dengan bahasa Python.

Konten seperti direktori dan file-file ditampilkan dalam bentuk kolom-kolom. Terdapat tiga buah kolom: kolom direktori sebelumnya, direktori sekarang, dan file preview atau direktori selanjutnya.
